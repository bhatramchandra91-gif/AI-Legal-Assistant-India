# ⚖️ AI Legal Assistant (India)

Complete AI tool for lawyers with:
- Drafting engine (court format based)
- OCR converter (Hindi/English/Marathi)
- ADR mediation AI
- Judgement & precedent scraper
- CRM for cases
- Improvement analyzer

## Tech Stack
Python, Streamlit, Playwright, OCR, OpenAI

## Run Locally
pip install -r requirements.txt
playwright install
streamlit run app.py

## Modules
1. Drafting Engine
2. OCR Converter
3. ADR Negotiator
4. Judgments & Precedents
5. CRM Dashboard
6. Improvement AI
